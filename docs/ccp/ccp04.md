# 第四章 存储器

### 4.1 概述

一、存储器分类

1. 按存储介质分类
    1. 半导体存储器     TTL、MOS
    2. 磁表面存储器     磁头、载磁体
    3. 磁芯存储器       硬磁材料、环状元件
    4. 光盘存储器       激光、磁光材料
   
    **其实是按半导体和磁材料，半导体有易失性、磁材料非易失性**

2. 按存取方式分类
   1. 存取时间与物理地址无关（随机访问）
       + 随机存储器：在程序的执行过程中可读可写
       + 只读存储器：在程序的执行过程中只读
    2. 存取时间与物理地址有关（串行访问）
       + 顺序存取存储器  磁带
       + 直接存取存储器  磁盘
    3. 按在计算机中的作用分类
